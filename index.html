Advanced Website Analytics

This page is actively tracked using a lightweight, privacy-friendly analytics script built with pure JavaScript.

Tracked Automatically:
  • Visitor ID (browser-persistent)
  • Session start time
  • Heartbeat every 10 seconds
  • Page URL changes
  • City (IP-based, once per session)

Why This Matters

This analytics approach is designed for:

  • No cookies required
  • No external libraries
  • Minimal CPU & memory usage
  • Server-side aggregation ready
  • Scales well on low-resource servers
